# **App Name**: Cuidar.me Lite

## Core Features:

- Boas-Vindas Automatizadas: Mensagens de boas-vindas automatizadas via WhatsApp para novos pacientes.
- Acompanhamento Programado: Mensagens de acompanhamento programadas via WhatsApp com base nos protocolos de tratamento do paciente.
- Sistema de Alerta de IA: Ferramenta de IA analisa as mensagens dos pacientes para detectar riscos potenciais, notificando os profissionais de saúde sobre alertas críticos.
- Extração de Dados: IA extrai dados estruturados (peso, níveis de glicose, etc.) das conversas dos pacientes para análise.
- Respostas Sugeridas: IA fornece rascunhos de respostas para os profissionais de saúde para acelerar as comunicações.
- Educação em Vídeo (LMS): Um módulo de educação em vídeo oferece aulas sobre dieta, uso de medicamentos, atividade física, gerenciamento de estresse e motivação.
- Lembretes de Aulas e Feedback: IA envia lembretes automáticos sobre novas aulas em vídeo e coleta feedback após a visualização.

## Style Guidelines:

- Cor primária: Azul suave (#75A4FF) para representar confiança, saúde e tranquilidade.
- Cor de fundo: Azul muito claro (#EAF2FF) para uma sensação limpa e calmante.
- Cor de destaque: Verde-azulado suave (#46B1BF) para destacar informações importantes e CTAs sem sobrecarregar o usuário.
- Fonte do corpo e do título: 'Inter' (sans-serif) para uma aparência moderna e limpa. Fácil de ler na tela para textos longos e exibição de informações.
- Use ícones consistentes e minimalistas em todo o aplicativo para ajudar na navegação e para representar recursos principais, como alertas, módulos de vídeo e entradas de dados.
- Enfatize um layout limpo e intuitivo. O conteúdo deve ser facilmente escaneável, com hierarquia e espaçamento claros. Garanta a legibilidade em diferentes dispositivos, principalmente em dispositivos móveis.
